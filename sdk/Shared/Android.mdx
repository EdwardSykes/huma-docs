# Android
Library that contains set of shared data structures and logic

### Getting Started
<Tabs>
<TabItem value="installing" label="Installing">

  1. Add the dependency in your local build.gradle file:

  ```
  implementation("com.huma.sdk:shared:latest-version")
  ```

  2. `HumaSharedManager` require dependencies to be initialized before it in `Application` class

  ```kotlin
  HumaUtilsManager.init(/*..*/)
  HumaCachingManager.init(/*..*/)
  HumaSharedManager.init(/*..*/)
  ```

</TabItem>
<TabItem value="readme" label="Readme" default>

  **For internal use only**
