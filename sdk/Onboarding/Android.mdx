# Android
Library that implements onboarding flow

### Getting Started
<Tabs>
<TabItem value="example" label="Example" default>

1. Initialize HumaOnboardingManager with a list of steps
```kotlin
HumaOnboardingManager.init(
   onboardingSteps: List<OnboardingStep>
)
```
2. Start OnboardingActivity
```kotlin
startActivity(OnboardingActivity.getIntent(context))
```

</TabItem>
<TabItem value="installing" label="Installing">

1. Add the dependency in your local build.gradle file:
```kotlin
implementation("com.huma.sdk:onboarding:<latest-version>")
```
2. Initialize `HumaOnboardingManager` in your Application class:
```kotlin
HumaOnboardingManager.init(
   onboardingSteps: List<OnboardingStep>
)
```

</TabItem>
<TabItem value="readme" label="Readme" default>

**OnboardingStep** - interface that onboarding flow step is required to implement. 
A list of steps is provided during initialization (note that order is important, conditions are checked sequentially):

```kotlin
HumaOnboardingManager.init(listOf(
            object : OnboardingStep {
                // will be shown if this condition passes
                override fun shouldBeShown(params: OnboardingStepParams): Boolean {
                    // params contains deployment, prefs manager and user instance to determine whether step should be shown
                    return true
                }

                // used to launch the step
                override fun getActivityIntent(context: Context): Intent {
                    return PreferredUnitsActivity.getLauncherIntent(
                        context,
                        PreferredUnitsActivity.Params.OnboardingMode
                    )
                }

                // used for logging purposes
                override fun getAnalyticsName() = "step name"
            }
        ))
```
